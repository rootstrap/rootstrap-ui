# Dispay

Changes the value of the [display property](https://developer.mozilla.org/en-US/docs/Web/CSS/display) with our responsive display utility classes

[Development file](https://github.com/rootstrap/rootstrap-ui/tree/develop/src/styles/utilities/_display.scss)

The classes are named using the format:

- `.d-{value}`
- `.d-{breakpoint}-{value}`

Where value is one of:

- `none`
- `inline`
- `inline-block`
- `block`
- `table`
- `cell`
- `table-cell`
- `table-row`
- `flex`
- `inline-flex`

## Customize display utilities

The display values can be altered by changing the `$displays` variable before importing **Rootstrap UI**'s styles.
